In general I make all of my code available through Github.
Some specific projects that I am working on/contributing to include:

Massive Online Analysis (MOA)
Open-source framework for data stream mining.
I have contributed code for data stream clustering algorithms (notably D-Stream) as well a collection of one-class classifiers. The Mixture Model data stream generators are also compatible with MOA 17.06. Project Github

Cribbage for Reinforcement Learning
Intelligent agents learning to play the classic card game
Project consists of the environment, agents, and a variety of training scripts. Github

csvCombiner
Utility for combining multiple databases into one CSV file.
Future work to increase the number of file formats supported and improve the UI. Github
